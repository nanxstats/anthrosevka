About

Origin and intent, kept brief.

Anthrosevka Mono is a custom build of Iosevka, configured for programming and terminal use. The build favors terminal spacing, regular and bold weights, a 600 width default, and unhinted TTF files for high-resolution displays.

Its visual direction is inspired by Anthropic Mono, with a narrow, clear, calm texture for code. It is an independent third-party project and does not imply endorsement, affiliation, or support from Anthropic PBC.

The aim is pragmatic: readable columns, distinguishable punctuation, stable symbols, and a tone that stays quiet during long coding sessions.

Usage / Download

Prebuilt files are published through GitHub releases. Use the local web font files directly when serving the specimen or a project page.
