# Anthrosevka Mono

## Build

Follow [building Iosevka from Source
](https://github.com/be5invis/Iosevka/blob/main/doc/custom-build.md)
to set up the build environment.

Then, put `private-build-plans.toml` in the root of the Iosevka clone. Run

```bash
npm run build -- ttf-unhinted::IosevkaCustom
```
