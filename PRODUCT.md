# Product

## Register

brand

## Users

Programmers, terminal users, and typeface maintainers evaluating a monospace font for coding and terminal work. They need to inspect glyph ambiguity, code texture, spacing, weights, and installation details quickly.

## Product purpose

Anthrosevka Mono is a custom build of Iosevka inspired by the look and feel of Anthropic Mono. The project provides prebuilt font files and source configuration for people who want a narrow, clear, calm programming font.

## Brand personality

Precise, restrained, practical. The tone should feel like a well-made open-source artifact: direct, typographic, technically literate, and legally cautious.

## Anti-references

Avoid SaaS launch pages, fake gradients, noisy animation, AI startup visuals, hype copy, overdesigned cards, and claims that imply affiliation with Anthropic or Iosevka beyond accurate source attribution.

## Design principles

- Make the typeface the main visual material.
- Use specimen content that helps evaluate programming typography, including ambiguous glyphs, punctuation, operators, brackets, digits, and realistic code.
- Keep copy concise, factual, and cautious.
- Prefer quiet rhythm, alignment, and contrast over decorative effects.
- Keep implementation static, dependency-free, accessible, and maintainable.

## Accessibility & inclusion

Respect `prefers-color-scheme` and `prefers-reduced-motion`. Preserve keyboard access for controls and tabs. Use readable contrast, visible focus states, semantic structure, and graceful font fallbacks.
